## ⚙️ 配置说明

### 内容配置
在 `index.html` 中找到 `LETTER_CONFIG` 对象：

```javascript
const LETTER_CONFIG = {
    artistName: '尹毓恪',        // 歌手署名
    title: '致粉丝的信',          // 信标题
    subtitle: 'A Letter For You', // 副标题
    paragraphs: [ /* ... */ ],    // 正文数组
    signaturePrefix: '永远爱你的', // 落款前缀
    signatureDate: '2025.12.30',  // 落款日期
    footerEasterEgg: '彩蛋文字'   // 页脚彩蛋
};
```

### 段落格式
```javascript
paragraphs: [
    // 普通段落
    '这是一段文字...',
    
    // 引用/诗句
    {
        type: 'quote',
        content: '诗句内容<br>可以换行'
    }
]
```

### 配色方案
在 CSS 中全局替换颜色代码：

```
当前配色（复古蓝灰）:
#4a6b7c - 主色调
#7a94a8 - 次要色
#9fb4c7 - 强调色
#d5e5ea - 背景色
```

## 🔒 数据存储

### localStorage 使用
**键名**: `letter_open_count`  
**值**: 打开次数（整数）  
**触发时机**: 点击信封成功拆封时  
**持久化**: 浏览器本地存储

### 重置方法
```javascript
// 浏览器控制台运行
localStorage.removeItem('letter_open_count');
// 或
resetOpenCount();
```

## 🎓 技术亮点

### CSS
1. **纸张纹理效果**
   - 使用 `repeating-linear-gradient` 创建网格纹理
   - 叠加多层渐变实现复古质感

2. **邮票锯齿边缘**
   - `radial-gradient` + `background-size` 模拟锯齿
   - `::before` 伪元素分层

3. **3D 信封效果**
   - `perspective` + `transform` 创建立体感
   - `clip-path` 裁剪三角形翻盖

### JavaScript
1. **localStorage 计数**
   - 防重复点击
   - 数字滚动动画

2. **Intersection Observer**
   - 段落进入视口时触发动画
   - 性能优化的滚动监听

3. **动态内容渲染**
   - 配置对象驱动
   - 支持多种段落类型

### SVG
1. **手绘风格插图**
   - 纯代码绘制，无外部图片
   - 路径和图形组合

2. **动画优化**
   - CSS 动画而非 JS 动画
   - `transform` 硬件加速
